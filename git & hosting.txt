git :

كيف تفتح مستودع جديد و اضافة الملفات في  

git init

git add . // يوجد مسافة ونقطة بعد كلمة add

git commit -m "first commit"

git branch -M main

git remote add origin اكتب الرابط هنا

git push -u origin main

---------------------------------------------------------------------------

firebase:

   كيف ترفع ملفاتك على السيرفر و تثبت الفاير بيس لأول مره
  

تثبيت اداة السيرفر npm install -g firebase-tools

تسجيل الدخول firebase login 

تحديد مجلد المشروع cd مسار المشروع

انشاء مستودع firebase init

انقل ملفات المشروع الى مجلد public

اذا اردت اختبار السيرفر قبل النشر  firebase serve 

نشر المشروع و تشغيل السيرفر firebase deploy
